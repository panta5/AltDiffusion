# AltDiffusion for EasyStableDiffusion

이 레포지토리는 EasyStableDiffusion을 코랩 이외의 GPU호스팅에서 쉽게 실행할 수 있게 돕도록 만들어졌습니다.

## 세 줄 요약

1\. 레포지토리 clone 및 실행권한 부여

```bash
git clone https://github.com/panta5/AltDiffusion.git
cd AltDiffusion
sudo chmod +x setup.sh
```

2\. setup\.sh 실행

```bash
sudo ./setup.sh
```

3\. start\.sh 실행

```bash
sudo ./start.sh
```
